Questão 2

Qual padrão de nomenclatura normalmente se usa no JS?

lowerCamelCase - Os parâmetros devem começar com letra minúscula e para cada palavra a seguir, a primeira letra deve ser maiúscula, por exemplo "codigoDoCliente".

Questão 3

Quando um valor de uma variável será undefined?

Quando a variável não possui um valor definido, como var b por exemplo, a variável apenas possui uma identificação b, mas não tem nenhum valor atribuído.

Questão 4

Pesquisa quando uma variável terá um valor NaN. O que é?

A propriedade global NaN é um valor especial que significa Not-A-Number (não é um número), caracteriza-se por uma expressão matemática que deu errado como, por exemplo, quando tentamos converter uma string para um int.
